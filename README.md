### Instalação
1. Instalar o Node: Versão LTS

#### Rodar API
1. Acessar pasta: `cd api`
2. npm install
3. npm run start
-------
### Rodar Front-end
1. Acessar pasta cd app
2. npm install
3. npm run dev

------

Acessar ao projeto:
http://localhost:3333 - Front-end
http://localhost:3000/ - Back-end

-----

OBS: Não está relacionado a nem um banco de dados.
